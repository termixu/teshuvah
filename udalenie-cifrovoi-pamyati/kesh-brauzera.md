### 🔍 Что такое кэш браузера

* Временные файлы, которые браузер сохраняет для ускорения загрузки сайтов
* Включает куки, изображения, скрипты, данные форм и автозаполнения
* Через кэш система зверя может хранить следы твоей активности и личные данные

***

### 🛠 Пошаговое удаление кэша в популярных браузерах

#### Google Chrome

* Меню → Настройки → Конфиденциальность и безопасность → Очистить данные просмотра
* Выбрать «Файлы cookie и другие данные сайтов» и «Изображения и файлы в кэше»
* Нажать «Очистить данные»

#### Mozilla Firefox

* Меню → Настройки → Конфиденциальность и безопасность → Куки и данные сайтов → Очистить данные
* Выбрать «Куки» и «Кэш»
* Подтвердить удаление

#### Safari (Mac)

* Safari → Настройки → Конфиденциальность → Управлять данными веб-сайтов → Удалить все
* Или: Safari → Очистить историю

#### Microsoft Edge

* Настройки → Конфиденциальность, поиск и службы → Очистить данные браузера → Выбрать данные для удаления → Очистить
