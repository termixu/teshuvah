# 🧹 Чистка кэша

🧹 **Чистка кэша и временных файлов** в macOS — это как генеральная уборка: быстро освобождает память, избавляет от багов и делает систему шустрее. Вот **полный, техничный, но понятный гайд**, чтобы ты мог сделать это вручную и осознанно, без сторонних программ. Всё — через Терминал (или Finder, если хочешь).

***

### 🗂️ Где живёт кэш?

Основные папки, в которых приложения хранят мусор:

```
~/Library/Caches
/Library/Caches
~/Library/Application Support
~/Library/Containers
```

`~` — означает твою домашнюю папку (`/Users/твоеимя`)

***

### 🔥 Удалить весь кэш пользователя (без риска для системы)

```bash
rm -rf ~/Library/Caches/*
```

💡 Это безопасно. Сюда пишут кэш обычные приложения, вроде браузеров, редакторов и т. д.

***

### 🔥 Очистить системный кэш (будет нужно ввести пароль)

```bash
sudo rm -rf /Library/Caches/*
```

⚠️ Не трогай `/System/Library` — это защищённая часть macOS, и с Ventura/Monterey всё под SIP.

***

### 📦 Удалить временные файлы приложений (Application Support)

Найди конкретную программу (пример: VSCode) и очисти её мусор:

```bash
rm -rf ~/Library/Application\ Support/Code/*
```

> Можешь посмотреть размер через `du -sh ~/Library/Application\ Support/Code`

***

### 🧬 Очистить Saved States (открытые окна и сессии)

```bash
rm -rf ~/Library/Saved\ Application\ State/*
```

***

### 🧼 Очистить временные логи и отчёты сбоев

```bash
rm -rf ~/Library/Logs/*
rm -rf ~/Library/DiagnosticReports/*
```

***

### 💽 Очистить кэш конкретного приложения

Хочешь удалить только у Safari?

```bash
rm -rf ~/Library/Caches/com.apple.Safari
```

Или у Telegram:

```bash
rm -rf ~/Library/Caches/org.telegram.desktop
```

***

### 👁‍🗨 Посмотреть, сколько места занимает кэш

```bash
du -sh ~/Library/Caches/*
```

📦 Получишь список всех папок кэша и их размер.

***

### 🧠 Как часто чистить?

* 📆 Раз в 1–2 недели — если активно пользуешься Mac
* ⚙️ После удаления тяжёлых программ
* 🚑 Если что-то тормозит или глючит

***

### ⚠️ Внимание:

* **Не трогай `/System`, SIP-защищённые разделы и `.kext`**
* **Чистка кэша может вызвать повторную загрузку ресурсов** (например, иконки, настройки редактора, индексация Spotlight)
