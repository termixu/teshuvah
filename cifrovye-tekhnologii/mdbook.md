# 📚 mdBook

### ✅ Установка mdBook:

#### 1. Установка mdBook

Если Rust у тебя ещё не установлен:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Затем установи mdBook:

```bash
cargo install mdbook
```

***

#### 2. Инициализация проекта

```bash
mdbook init pilgrim-book
cd pilgrim-book
```

Это создаст структуру с нужными файлами.

***

#### 3. Структура и запуск

* Отредактируй файл `SUMMARY.md` — туда добавим оглавление книги.
* Пиши главы в папках `src/`.
* Запусти локальный сервер для просмотра:

```bash
mdbook serve
```

Или собери статический сайт:

```bash
mdbook build
```

***

#### 4. Настройка внешнего вида

mdBook поддерживает темы и кастомизацию CSS, можно будет сделать оформление под паломнический стиль.

***

### ✅ Что ты сможешь делать офлайн:

#### 📖 Читать книгу

* После `mdbook build`, в папке `book/` будет готовый **статический сайт**.
* Открываешь файл `book/index.html` в любом браузере — работает как полноценный сайт.
* Весь интерфейс (оглавление, поиск, прокрутка, тёмная тема и т.д.) будет работать **без интернета**.

#### 🧑‍💻 Редактировать главы

* Просто правишь `.md` файлы в `src/` — это обычный Markdown.
* Можно использовать любой текстовый редактор: VSCode, Sublime, Gedit, даже `nano`.

#### 🏗️ Генерировать новую версию

* Команда `mdbook build` всегда работает локально.
* Не требует ни интернета, ни подключения к GitHub.

#### 🔍 Поиск и навигация

* Встроенный **поиск** в mdBook работает офлайн через встроенный JavaScript.
* Навигация, меню, ссылки — всё также работает.

***

### 📦 Что нужно будет установить только один раз:

1. **Rust** (однократно)
2. **mdBook** (через `cargo install mdbook`)

> После этого ты полностью автономен: можешь создавать, редактировать и читать свои книги **в любое время и в любом месте** — даже в глуши, даже без электричества, если есть ноутбук на батарее.
