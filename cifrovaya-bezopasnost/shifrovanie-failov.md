# 🗄️ Шифрование файлов

> 📌 Шифрование — это способ скрыть содержимое файла от посторонних. Только тот, у кого есть ключ, сможет его расшифровать и прочитать.

***

### 🔐 Зачем шифровать файлы?

* Хранишь важные данные (записи, письма, документы);
* Передаёшь информацию по сети или физически (флешки, диски);
* Живёшь вне системы — шифрование защищает от кражи, конфискации, утечки;
* Чтобы **внешний человек не мог прочесть без твоего разрешения**.

### 📁 Шифрование одного файла (GPG)

```bash
gpg -c important.txt
```

* `-c` — symmetric encryption (симметричное шифрование);
* Создаётся `important.txt.gpg` — зашифрованная копия;
* Оригинал можно удалить.

Расшифровка:

```bash
gpg important.txt.gpg
```

### 📦 Шифрование архива (7-Zip)

```bash
7z a -t7z -p secret.7z my_folder/
```

* `-p` — запрашивает пароль;
* `-t7z` — формат 7z с AES-256.

💡 Отличный способ создать зашифрованный контейнер, особенно на Windows.

***

### 🔐 Шифрованный том (VeraCrypt)

* Создаётся **виртуальный диск**, который монтируется как флешка;
* Можно переносить на флешках, использовать как "скрытую папку".

🔧 Можно скрыть сам факт наличия зашифрованных данных (стеганография + скрытые тома VeraCrypt).

***

### 🔄 Автоматическое шифрование/дешифрование

Для более продвинутых:

* Используй **Cryptomator**, если работаешь с облаками;
* **EncFS**, **gocryptfs** — для прозрачного шифрования папок;
* **age + sops** — для конфигураций (в dev-средах).

***

### 🔥 Безопасное удаление оригинала

После шифрования **удали оригинал** так, чтобы он не мог быть восстановлен:

```bash
shred -u important.txt
```

* `shred` — перезаписывает файл;
* `-u` — удаляет после.

***

### 📖 Христианская суть

> 💬 _"Будь мудр, сын мой, и возвесели сердце моё..."_ (Прит. 27:11)

Шифрование не для страха, а для чистоты, тайны и свободы. Ты не прячешь грех — ты сохраняешь путь истины в закрытом сосуде. Пусть твоя цифровая жизнь будет как запечатанная книга, открытая только Господу и достойным.

***

### 🧰 Быстрая памятка (CLI)

| Действие          | Команда                      |
| ----------------- | ---------------------------- |
| Шифровать файл    | `gpg -c file.txt`            |
| Расшифровать файл | `gpg file.txt.gpg`           |
| Удалить безопасно | `shred -u file.txt`          |
| Архив с паролем   | `7z a -p archive.7z folder/` |
| Создать GPG ключ  | `gpg --full-generate-key`    |
