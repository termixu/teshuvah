### 🎯 Цель модуля

* 🔥 Полностью стереть своё цифровое имя в Вавилоне
* ❌ Разрушить цифровой образ (ID), чтобы не быть привязанным к системе
* 🕊 Оставить только то, что действительно необходимо — паспорт и вера

### 🛠 Инструкция по удалению популярных сервисов

#### 🏛 Госуслуги

1. Войдите в [gosuslugi.ru](https://www.gosuslugi.ru/)
2. Перейдите: Настройки → Удаление учётной записи
3. Подтвердите вход по СМС
4. Отказ от восстановления
5. Готово

#### ☁️ iCloud / Apple ID

1. Зайди на [privacy.apple.com](https://privacy.apple.com/)
2. Войти и выбрать «Удаление учётной записи»
3. Получи код, подтверди
4. Удаление занимает до 7 дней
5. Устройство отвяжется при удалении

#### 🎮 Steam

1. Зайти в профиль
2. Настройки → Удалить аккаунт
3. Следовать инструкциям, подтвердить почтой
4. Не входить 30 дней — и аккаунт исчезнет

#### 💼 Airbnb

1. [airbnb.com](https://www.airbnb.com/) → Настройки аккаунта
2. Вниз → _«Удалить аккаунт»_
3. Подтвердить по почте
4. Безвозвратное удаление
