### ☁️ Что такое iCloud

* Облачное хранилище Apple, где сохраняются:
  * фотографии
  * контакты
  * заметки
  * резервные копии устройств
  * пароли и другие личные данные
* iCloud связан с твоим Apple ID и даёт системе зверя доступ к информации

***

### 🛠 Пошаговое удаление из iCloud

#### 1. Отключение синхронизации на устройстве

* Настройки → Apple ID → iCloud
* Отключить все пункты: Фото, Контакты, Календарь, Заметки, iCloud Drive и т.д.

#### 2. Удаление данных из iCloud через устройство

* Настройки → Apple ID → iCloud → Управление хранилищем
* Удалить резервные копии устройств
* Удалить документы и данные приложений

#### 3. Удаление данных с сайта iCloud

* Перейти на [icloud.com](https://icloud.com/)
* Войти под своим Apple ID
* Удалить все файлы: Фото, Почту, Заметки, Контакты, Календарь и т.д.

#### 4. Запрос удаления Apple ID (если нужно)

* Перейти на [privacy.apple.com](https://privacy.apple.com/)
* Войти → Запросить удаление аккаунта
* Следовать инструкциям для полного стирания данных и аккаунта
