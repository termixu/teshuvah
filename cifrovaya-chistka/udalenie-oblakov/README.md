### 🛠 Инструкции по удалению данных

#### ☁️ iCloud

1. Отключить в настройках iPhone синхронизацию iCloud
2. Зайти на [icloud.com](https://icloud.com/) → Файлы и Фото
3. Удалить все документы, файлы, фото
4. Очистить корзину iCloud
5. Выйти из Apple ID (если возможно)

#### ☁️ Google Drive

1. Перейти на [drive.google.com](https://drive.google.com/)
2. Выделить все файлы и удалить
3. Очистить корзину
4. Удалить аккаунт Google (если в рамках плана)

#### ☁️ Dropbox, OneDrive, Яндекс.Диск

* Аналогично: удалить все файлы, очистить корзину, деактивировать аккаунт
