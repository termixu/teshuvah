# Системные настройки

### 🍏 macOS — основные шаги

* **Удаление аккаунтов:**\
  Системные настройки → Пользователи и группы → Выбери ненужный аккаунт → Удалить
* **Отключение iCloud:**\
  Системные настройки → Apple ID → Выйти из iCloud (с удалением данных с Mac)
* **Удаление ключей и паролей:**\
  Приложение "Связка ключей" → Удалить все записи
* **Отключение автоматической синхронизации:**\
  Системные настройки → Apple ID → iCloud → Отключить все пункты
* **Удаление приложений и связанных данных:**\
  Удалить приложения через Launchpad или Finder + очистить \~/Library и \~/Library/Application Support

***

### 🪟 Windows — основные шаги

* **Удаление аккаунтов:**\
  Параметры → Учетные записи → Семья и другие пользователи → Удалить лишние аккаунты
* **Отключение синхронизации:**\
  Параметры → Учетные записи → Синхронизация → Отключить синхронизацию параметров
* **Очистка данных учетной записи Microsoft:**\
  Выйти из Microsoft-аккаунта, использовать локальную учетную запись
* **Удаление паролей и данных автозаполнения:**\
  Панель управления → Учетные записи пользователей → Управление учетными данными → Удалить сохранённые пароли
* **Удаление приложений:**\
  Параметры → Приложения → Удалить ненужные программы

***

### 🔕 Общие рекомендации

* Выключить автоматические обновления и синхронизацию с облаком
* Убедиться, что все аккаунты и данные удалены или отключены
* Сделать полную проверку настроек безопасности и конфиденциальности
