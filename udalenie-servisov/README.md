### 🎯 Цель модуля

* 🔥 Удалить учётные записи в главных цифровых империях мира

* 💡 Выйти из цифровых экосистем, которые формируют мышление и привычки

### 🛠 Инструкция по удалению (Google / Apple / Яндекс)

#### 📍 Google:

1. Перейди: [myaccount.google.com](https://myaccount.google.com/)
2. Вкладка «Данные и конфиденциальность»
3. Низ страницы — _«Удаление аккаунта Google»_
4. Подтверди по почте и паролю
5. Молись об освобождении

#### 📍 Apple:

1. Перейди: [privacy.apple.com](https://privacy.apple.com/)
2. Войти в аккаунт
3. Нажми _«Запросить удаление учётной записи»_
4. Пройди проверку
5. Получи код удаления, запиши
6. Ожидай 7 дней

#### 📍 Yandex:

1. Перейди в [паспорт Яндекса](https://passport.yandex.ru/profile)
2. Внизу — _«Удалить аккаунт»_
3. Вводишь пароль, подтверждаешь удаление
