# 📄 Удаление документов

### 📖 Основание из Писания

> «Забудь народ твой и дом отца твоего, и возжелает Царь красоты твоей»\
> 📖 **Псалом 44:11**

> «Мы не от мира сего…»\
> 📖 **Иоанна 17:14–16**

***

### 🎯 Цель модуля

* 🔥 Избавиться от ненужных **цифровых и бумажных документов**, созданных для контроля, слежки и управления
* ✂️ Удалить всё, что ассоциирует тебя с системой мира и цифровым зверем
* 🕊 Оставить только то, что пока необходимо — _для переходного периода или выхода_

***

### 📃 Что считать документом зверя?

| 📂 Документ                              | ⚠️ Причина удаления                           |
| ---------------------------------------- | --------------------------------------------- |
| 🧾 ИНН (идентификационный номер)         | Цифровая метка личности, налоговая привязка   |
| 📋 СНИЛС                                 | Цифровое страхование, мед. привязка к системе |
| 📛 Полис ОМС                             | Медицинский контроль, обязательность          |
| 📜 Аттестаты / дипломы                   | Привязка к системе образования Вавилона       |
| 📕 Трудовая книжка                       | Привязка к "работе от мира", цифра работника  |
| 🧾 Договора (банки, кредиты, аренда)     | Цепи долга, финансовые метки                  |
| 📬 Регистрации / прописки                | Местоположение, юрисдикция, контроль          |
| 🎟️ Карты пенсионера / студента / и т.д. | Социальные статусы, часть системы             |

***
